What is Palindrome number?

A palindrome number is a number that remains the same even its digits are reversed.
Like 121, 161 , 12321.
Programmatic approach to see wether number is palindrome or not.
1 >> we take number as input.
2 >> then store input in temp variable.
3 >> revers the input.
4 >> check number is palindrome or not
5 >> print results .

  • C
  • JAVA
  • More 

 #include <stdio.h>
int main()
{ 
    int inputNum ,temp, digit, reverse = 0;
    
    printf("Enter a number to check if it is a palindrome or not\n");
    scanf("%d",&inputNum);
    temp = inputNum; //storing number in temp variable
    while (inputNum != 0)
    { 
        digit = inputNum % 10;
        reverse = (reverse * 10 ) + digit;
        inputNum = inputNum/10;
    }
    // checks is input and reversed number is same or not
    if(temp == reverse){
        printf("%d is a palindrome number.\n", temp);

    }
    else {
        printf("%d isn't a palindrome number.\n", temp);
        
    }
     return 0;
}

What is Fibonacci Serries?

The Fibonacci is a Serries of numbers, each number (term) is the sum of the previous two numbers. like 0 , 1, 1, 2, 3, 5, ...
Programmatic approach to achive fibonacci Series up to number of terms.
1 >> Take number of terms.
2 >> loop upto term.
3 >> inside loop add sum of two previous terms to next term.
4 >> print results.

  • C
  • JAVA
  • More 

#include 
int main()
{
    int i, input, term1 = 0, term2 = 1, nextTerm;

    printf("Enter the number of terms: ");
    scanf("%d", &input);

    printf("Fibonacci Series: ");

    for (i = 1; i <= input; ++i)
    {
        printf("%d, ", term1);
        nextTerm = term1 + term2;
        term1 = term2;
        term2 = nextTerm;
    }
    return 0;
}

What is Factorial number?

Factorial is a number N which is product of all postive number upto N.
like 5 = 1 * 2 * 3 * 4 * 5 = 120 .
Programmatic approach to find factorial of a number .
1 >> Take input from user.
2 >> Factorial initialization to 1.
3 >> run loop upto N .
4 >> multiply the current loop counter value.

  • C
  • JAVA
  • More 
    
 #include 

int main()
{
    int i, num;
    unsigned long long fact=1LL;

    //Input number from user
    printf("Enter any postive number to calculate factorial: ");
    scanf("%d", &num);

     // looping 1 to num 
    for(i=1; i<=num; i++)
    {
        fact = fact * i;
    }

    printf("Factorial of %d = %llu", num, fact);

    return 0;
}

What is Bubble sort?

Bubble sort is a algorith for sorting elements in array. It traversed from first to last element and compares current element with the next and swap with largest.
Programmatic approach to sort array using bubble sort algorithm.
1 >> Create array .
2 >> loop array upto array length.
3 >> compare current element with next element in array.
4 >> swap elements.
5 >> print sorted array.

  • C
  • JAVA
  • More 

#include 
 
void swap(int *xp, int *yp)
{
    int temp = *xp;
    *xp = *yp;
    *yp = temp;
}
 
// A function to implement bubble sort
void bubbleSort(int arr[], int n)
{
   int i, j;
   for (i = 0; i < n-1; i++)      
 
       // Last i elements are already in place   
       for (j = 0; j < n-i-1; j++) 
           if (arr[j] > arr[j+1])
              swap(&arr[j], &arr[j+1]);
}
 
/* Function to print an array */
void printArray(int arr[], int size)
{
    int i;
    for (i=0; i < size; i++)
        printf("%d ", arr[i]);
    printf("n");
}
 
// Driver program to test above functions
int main()
{
    int arr[] = {64, 34, 25, 12, 22, 11, 90};
    int n = sizeof(arr)/sizeof(arr[0]);
    bubbleSort(arr, n);
    printf("Sorted array: \n");
    printArray(arr, n);
    return 0;
}



 public class BubbleSortExample {  
    static void bubbleSort(int[] arr) {  
        int n = arr.length;  
        int temp = 0;  
         for(int i=0; i < n; i++){  
                 for(int j=1; j < (n-i); j++){  
                          if(arr[j-1] > arr[j]){  
                                 //swap elements  
                                 temp = arr[j-1];  
                                 arr[j-1] = arr[j];  
                                 arr[j] = temp;  
                         }  
                          
                 }  
         }  
  
    }  
    public static void main(String[] args) {  
                int arr[] ={3,60,35,2,45,320,5};  
                 
                System.out.println("Array Before Bubble Sort");  
                for(int i=0; i < arr.length; i++){  
                        System.out.print(arr[i] + " ");  
                }  
                System.out.println();  
                  
                bubbleSort(arr);//sorting array elements using bubble sort  
                 
                System.out.println("Array After Bubble Sort");  
                for(int i=0; i < arr.length; i++){  
                        System.out.print(arr[i] + " ");  
                }  
   
        }  
}

FIND THIRD LARGEST NUMBER IS ARRAY?

We can find the third largest number in an array in java by sorting the array and returning the 3nd largest number.
Programmatic approach to find third largest number in array.
1 >> Take array or declare.
2 >> sort the array .
3 >> then get third largest number by index. 4 >> print results.

  • C
  • JAVA
  • More 

#include 
int getThirdLargest(int a[], int total){
int temp;  
for (int i = 0; i < total; i++)   
        {  
            for (int j = i + 1; j < total; j++)   
            {  
                if (a[i] > a[j])   
                {  
                    temp = a[i];  
                    a[i] = a[j];  
                    a[j] = temp;  
                }  
            }  
        }  
       return a[total-3]; 
}
int main()
{
    int a[]={1,2,5,6,3,2};  
    
    printf("Third Largest: %d "+getThirdLargest(a,6));  
  
}
